# 同源与跨域

## 概念

 - 同源
        http://127.0.0.1/2017-8-17/09.waterFall_ajax/
        http://127.0.0.1/2017-8-17/09.waterFall_ajax/api/waterFall.php
        协议名:都是http
        地址：一样
        端口号:默认是80端口
         上述三个条件一样 称之为 同源
- 不同源
        协议,地址,端口号 有一个不一样 称之为不同源
- 跨域
        不同源的网站之前 互相发送请求
